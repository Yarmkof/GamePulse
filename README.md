# Yarmkof GamePulse ⚽

Application web sportive spéciale Coupe du Monde.

## Fonctionnalités

- Scores en direct fictifs
- Temps de jeu
- Buteurs
- Cartons
- Statistiques
- Détails de match
- Mode clair / sombre
- Compatible GitHub Pages

## Lancer en local

```bash
npm install
npm run dev
```

## Compiler

```bash
npm run build
```

## Publication GitHub Pages

Le dossier `dist` contient la version prête à publier.
